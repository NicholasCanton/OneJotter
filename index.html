可乐记

简单，快速的方式管理你的笔记

可乐记是一个极简风格设计的笔记产品，适合笔记记录和散文写作，它附带了一个功能丰富的扩展键盘和一些当下流行的主题，既可以作为纯文本编辑器使用又可以保存富媒体样式的笔记，同时还支持 Markdown 格式的编辑及预览。

扩展键盘

强大丰富的扩展键帮您快速实现美观易读的富媒体笔记。
  • 特殊字符
  • 拍照，视频或者从图库选择
  • 附件
  • 素绘
  • 标签
  • 标题
  • 粗体，斜体，下划线和删除线
  • 水平分割线
  • 清单
  • 列表
  • 引用
  • 代码
  • 代码块
  • 链接
  • 日期
  • 撤销，重做
  • 上移，下移
  • 缩进，取消缩进

功能特征

丰富的功能为您提供便捷与效率。
  • 搜索笔记
  • 同步笔记
  • 合并笔记
  • 打印笔记
  • 快速格式化笔记
  • 通知栏创建快速笔记
  • 编辑 Markdown 及预览
  • 富媒体与 Markdown 自由转换
  • 备份，还原笔记
  • 导入，导出笔记
  • 创建类别，文件夹和标签
  • 切换主题，字体
  • 指纹与密码解锁笔记
  • 还有更多...

云端存储

多种网络同步与存储方式，自由选择。
  • Google Drive
  • Microsoft OneDrive
  • WebDAV Server

导出与分享

导出多种格式的笔记，一键快速分享到其它应用。
  • TXT
  • JPEG
  • PDF
  • EPUB
  • HTML
  • Markdown
  • TextPack
