Personal notebook to save and organize your notes.

One Jotter is an easy-to-use notebook and notepad app perfect for organizing notes and recording ideas. It allows you to keep everything organized and avoid clutter with less effort. You can use it to edit plain text files or save notes in RTF format. It also supports writing and previewing in Markdown (CommonMark) format.

Take notes effortlessly, Create simple yet powerful note cards with rich content, images, videos, audio, checklists...


Key Features

Search, synchronize, merge, print, format notes, create notes from notification bar, edit and preview in Markdown format, convert notes between Markdown and RTF, backup, restore, import, export notes, and more...

Extended Keyboard

You can attach photos, videos, audios, sketch, and more. Also supports bold, italics, underline, strikethrough, headings, horizontal lines, bulleted lists, numbered lists, lists, block quotes, inline code, code blocks, and can be undo and redo...

Cloud Storage

Supports synchronization with Google Drive, Dropbox, Microsoft OneDrive and WebDAV Server. Your data only exists on your own device or synced cloud drive, under your control, and no one can see it.

Export Notes

You can save your notes in TXT, PDF, JPEG, EPUB, HTML, Markdown formats, or share them as note cards.
