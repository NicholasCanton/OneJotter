使用可乐记「One Jotter」整理笔记

可乐记「One Jotter」 是一款易于使用的笔记本和记事本应用程序，非常适合整理笔记和记录想法。 它可以帮助您保持井井有条，避免杂乱无章。您可以使用它编辑纯文本文件，也可以保存 RTF 格式的笔记，它还支持 Markdown（CommonMark）格式的书写和预览。

轻松记笔记，创建简单但功能强大的记事卡，其中包含丰富的内容、图像、视频、音频、清单等...


功能概览

搜索，同步，合并，打印，格式化笔记，通知栏创建笔记，编辑 Markdown 格式的笔记及预览，富媒体样式与 Markdown 格式自由转换，备份，还原笔记，导入，导出笔记等。

扩展键盘

支持插入特殊字符，照片，视频，文件，素绘，粗体，斜体，下划线，删除线，水平分割线，清单，列表，引用，代码，链接，日期等，还可以撤销，重做，段落上移，下移，缩进，取消缩进。

云端同步

支持 Google Drive，Dropbox，Microsoft OneDrive，WebDAV Server 等多种网络同步与存储方式。数据仅存在于您自己的设备或同步的云驱动器上，由您自己控制，没有人可以看到它。

导出分享

可以导出笔记为 TXT，JPEG， PDF，EPUB，HTML，Markdown，TextPack 等多种格式，支持快速分享到其它应用。

通知栏笔记

快速创建笔记，高效编辑分享。

标签

管理笔记，灵活有序。

深色模式

夜间保护双眼，开关随意控制。

密码锁定

保护隐私，安心创作。

布局样式

三种布局样式，每种各具特色。

精美的主题

丰富多彩的配色，为写作锦上添花。

下载应用

适用于 Android, Windows 等设备，随时随地编辑与同步您的笔记。


定价

可乐记「One Jotter」提供的绝大部分功能都是可以免费使用的，没有时间限制，高级版功能可在应用内购买（支持华为与谷歌）或者使用激活码。
